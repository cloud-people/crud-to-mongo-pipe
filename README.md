## crud-to-mongo pipe

### About
Creates mongo query actions from crud actions, with correct data.


### How to use
First install dependencies.
```bash
yarn
```
Build package.
```bash
yarn build
```
Run Tests.
```bash
yarn test
```
Get type docs.
```bash
yarn doc
```

### Actions
1. loadById
2. loadbyQuery
3. removeById
4. removeByQuery
5. create
6. updateById
7. updateByQuery
8. aggregate
